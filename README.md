# SpotifyDisplay
ESP32 Spotify Display

A small desktop device that displays your currently playing Spotify track on a TFT screen using an ESP32.
The device connects to WiFi, authenticates with the Spotify API, and shows the song title and artist in real time.

This project combines embedded programming, networking, SPI communication, and hardware design.

Displays current Spotify song
Shows artist name
Automatically updates when the track changes
